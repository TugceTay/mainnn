Database Comparer VCL demo programs for C++Builder

Database Comparer (ADO) Compares, synchronizes and updates databases structure. Connects to MSSQL database via ADO.
Database Comparer (FIBPlus) Compares, synchronizes and updates databases structure. Connects to IB database via FIBPlus.
This demo is not available starting from RAD Studio 10 Seattle.
Database Comparer (FireDAC) Compares, synchronizes and updates databases structure and table data. Connects to Firebird 3.0 database via FireDAC.
Available starting from RAD Studio XE8.
Database Comparer (IB Express) Compares, synchronizes and updates databases structure and table data. Connects to IB database via IBExpress.
Extract Metadata Extracts the database metadata and displays the script.
Execute Local SQL Script Loads and executes LocalSQL (BDE) script for local databases (Paradox, dBASE).
This demo is not available starting from RAD Studio XE7.
SQL Script Comparer Compares two database scripts and displays the resulting update script.
Table Data Comparer Compares and synchronizes data between two database tables.
This demo is not available starting from RAD Studio XE7.
Please check Database Comparer (FireDAC) and Database Comparer (IB Express) demos.
